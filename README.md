# URL Shortener

Java Backend pet-project для сокращения ссылок.

## Live demo

Frontend:

http://5.42.112.163/app/

Example short link:

http://5.42.112.163/6fac6a51

## Deployment

Проект задеплоен на VPS через Docker Compose и Nginx.

Запущены сервисы:

- shortener-service
- analytics-service
- PostgreSQL
- Redis
- Kafka
- Zookeeper
- Prometheus
- Grafana
